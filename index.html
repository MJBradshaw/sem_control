Sem control

Basic program for generating raster scans for scanning electron microscope and viewing and saving images.

    Welcome to GitHub Pages.

    This is a program written in python with a Tk gui for the control of a scanning electron microscope. It has the following features:

    • Generate a raster scan on a National Instruments NIDAQ 6.9 compatible device.
    • Generate a live image of from the microscope by recording the video signal during the raster scan.
    • Save the image data.

    The hardware used during the testing of this program is:

    • Amray AMR-1000 Scanning Electron Microscope
    • National Instruments DAQPAD-1200 12-bit DAQ board
    • National Instruments NIDAQ 6.9 driver on Windows XP
